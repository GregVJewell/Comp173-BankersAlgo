This is a Python project, for the easiest route, import into PyCharm or similar IDE.

Please note that you need to import the project, then put the file you wish to iterate through into the 'Input Files' directory!

After importing/storing the files you'd like to run, start the program through whichever method you have chosen, and follow the prompts.

Example:
	C:\Users\GregV\PyCharmProjects\DeadlockDetection>python BankerAlgo.py
	>> input1.txt
	....
	>> output shown
	>> Run Again?
	>> Y\N
